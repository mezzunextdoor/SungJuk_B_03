sequenceDiagram
    autonumber
    actor 교수 as 교수
    participant UI as 성적시스템화면
    participant Controller as 성적관리컨트롤러
    participant Data as 성적데이터

    Note over 교수, Data: [시나리오 1: 성적입력]
    
    교수->>UI: 성적 입력 요청(교수id, 학생id, 자바, DB, 보안)
    UI->>Controller: 성적 저장 요청(교수id, 학생id, 자바, DB, 보안)
    
    rect rgb(240, 240, 240)
        Note right of Controller: 교수체크(include)
        Controller->>Controller: 교수id 확인 ("inha" 여부)
    end

    alt 교수id가 "inha"인 경우
        Controller->>Controller: 총점 및 평균 계산
        Controller->>Data: 성적 정보 저장(학생id, 자바, DB, 보안, 총점, 평균)
        Data-->>Controller: 저장 완료
        Controller-->>UI: 입력 성공 메시지 반환
        UI-->>교수: 성적 입력 완료 화면 표시
    else 교수id 불일치
        Controller-->>UI: 권한 오류 메시지 반환
        UI-->>교수: "교수id를 확인하세요" 경고 표시
    end

    Note over 교수, Data: [시나리오 2: 학점조회]

    교수->>UI: 학점 조회 요청(교수id, 학생id)
    UI->>Controller: 학점 산출 및 조회 요청(교수id, 학생id)

    rect rgb(240, 240, 240)
        Note right of Controller: 교수체크(include)
        Controller->>Controller: 교수id 확인 ("inha" 여부)
    end

    alt 교수id가 "inha"인 경우
        Controller->>Data: 학생 성적 데이터 요청(학생id)
        Data-->>Controller: 성적 데이터 반환(평균 점수 포함)
        
        Note right of Controller: 학점 부여 로직
        Controller->>Controller: 학점 결정(A: 90이상, B: 80이상, C: 80미만)
        
        Controller-->>UI: 결과 반환(학생id, 평균, 학점)
        UI-->>교수: 학점 결과 화면 출력
    else 교수id 불일치
        Controller-->>UI: 권한 오류 메시지 반환
        UI-->>교수: 조회 권한 없음 안내
    end
